Генерирует текст песни, состоящей из лексикона русскоязычных исполнителей и негативных комментаторов(но можно обучать на любых других данных).
На вход требует соответствующую базу данных для обучения.
Пытается осмысленно связать слова, подбирает слова в рифму.
start - kunteynir.py, добавить в путь текстовый файл с каким-либо датасетом, состоящим из русских предложений.
